betabeers-mean
====================

Material de la charla Fullstack Javascript organizada para Betabeers Zaragoza

## Requisitos
Instalar mongodb
Instalar node
Instalar npm
Instalar bower

## Importar datos de ejemplo
mongoimport --db betabeers --collection busstops --type json --file bus.json

## Puesta en marcha
npm install
bower install
grunt serve
